# Salesforce Thirukural Automation 🚀

A 100% low-code solution that integrates ancient Tamil wisdom with modern cloud technology. This system autonomously fetches and delivers a daily Thirukural with explanations to users every morning.

## 🌟 Project Overview
* **Goal:** Automate the daily sharing of Thirukural.
* **Architecture:** Uses Salesforce Schedule-Triggered Flows.
* **Tech Stack:** Salesforce CRM, Flow Builder, and External JSON API.
* **Manual Effort:** 0% (Fully Automated).

## 🛠️ How It Works
1. **Trigger:** Salesforce Flow wakes up daily at 7:00 AM.
2. **Fetch:** Executes an HTTP Callout to an external Thirukural API.
3. **Process:** Formats the Tamil text, English translation, and meaning.
4. **Deliver:** Sends the content to the designated user records.

## 📈 Impact
* **Consistency:** Daily cultural engagement without human intervention.
* **Efficiency:** Saves manual effort and ensures timely delivery.
* **Skill:** Demonstrates expertise in API integration and CRM automation without custom coding.
